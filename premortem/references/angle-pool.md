# Pool of perspectives (углы зрения)

Скилл выбирает 6 углов под тип плана:
4 критичных + 1 ситуативный + «Будущий поддерживающий» (с WYSIATI-микрошагом).

## Base (почти всегда)
- **Клиент / пользователь** — что думает тот, для кого делается
- **Исполнитель / команда** — чего не хватит у тех, кто делает
- **Противник или альтернатива** — кто/что перебьёт; на что уйдёт пользователь вместо
- **Допущения и доказательства** — что считаем правдой без проверки
- **Будущий поддерживающий** — как это будет жить через год без активного внимания.
  Включает WYSIATI-микрошаг: «кого/что мы НЕ видим?»

## Ситуативные (один добавляется к базовым под тип плана)

| Тип плана | Доп. угол 1 | Доп. угол 2 |
|-----------|-------------|-------------|
| `product_launch` | Конкурент | Ценообразование |
| `tech_project` | Архитектура | Эксплуатация |
| `strategic_pivot` | Рынок | Тайминг |
| `personal_decision` | Будущий-ты-через-N-лет | (только 1 доп.) |
| `hire` | Культура команды | Runway |
| `pricing_change` | Существующие клиенты | Churn |
| `nfp_book` | Читательский опыт | Конкурентные книги |
| `default` | Конкурент | Тайминг |

## Профили (6 углов = 4 базовых + 2 ситуативных или 4+1+Future)

Всегда включены: Клиент, Исполнитель, Противник, Допущения.
Ситуативные — из таблицы выше.
«Будущий поддерживающий» — всегда 6-й угол (с WYSIATI).

3 помощника = (2 угла, 2 угла, 2 угла).
Будущий поддерживающий отдаётся одному из помощников вместе с другим углом.
